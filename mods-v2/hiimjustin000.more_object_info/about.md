# More Object Info
A mod that adds more object properties to the object info text in the level editor.

# Features
- Extra object properties in the object info selection text, which can be enabled or disabled in the mod's settings

# List of Properties
- Object ID
- Position (X, Y)
- Rotation (X, Y)
- Scale (X, Y)
- Base Color HSV
- Detail Color HSV
- Object Type
- Object Address
