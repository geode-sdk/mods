# Node IDs

Adds node ids to layers for other mods to base their UI on, for preserving mod compatibility. See the [Github repository](https://github.com/geode-sdk/NodeIDs) for contributing.

Has ids for:

- CommentCell
- CreatorLayer
- DailyLevelNode
- DailyLevelPage
- EditLevelLayer
- InfoLayer
- LeaderboardsLayer
- LevelBrowserLayer
- LevelCell
- LevelInfoLayer
- LevelLeaderboard
- LevelSearchLayer
- ProfilePage
- RewardsPage
